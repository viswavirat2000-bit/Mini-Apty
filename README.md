# Mini Apty (monorepo)

This repository contains the Mini Apty extension and backend. See `Mini_Apty/specs` for the product spec and plan.

Workspaces: `Mini_Apty/extension`, `Mini_Apty/backend`.

Use `pnpm` and run `pnpm -w -r install` then `pnpm -w -r run typecheck`.
# Mini-Apty
Digital Adoption Platform prototype built with a Chrome Manifest V3 extension and Node.js API, supporting walkthrough recording, playback, authentication, and persistence.
